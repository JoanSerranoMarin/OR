# Scripts associated with the TFM on human olfactory receptors

This repository contains the scripts used for the computational analyses presented in the TFM:

**Identification of differential and distinctive features in human olfactory receptors using AlphaFold models**  
Joan Serrano Marin, July 2026

The code is organized according to the methodological blocks and result sections of the report. The main correspondence between the folders and the TFM is:

| Repository folder | TFM section / result block | Purpose |
| --- | --- | --- |
| `scripts/01_bw_annotation_and_distances/` | Methods II.2-II.3; Results III.1 | Ballesteros-Weinstein / GPCRdb annotation, inter-residue distances and ionic-lock descriptors. |
| `scripts/02_ionic_lock_statistics_and_figures/` | Results III.1; Figures 1-2; Supplementary Figures 1-2 | Statistical analysis and figure preparation for canonical and alternative ionic-lock metrics. |
| `scripts/03_cavity_volume_analysis/` | Results III.4; Figures 3-5 | Cavity-volume comparisons: extracellular portal, EC vestibule, TM5-TM7 cleft, orthosteric pocket, hydrophobic layer and intracellular cavity. |
| `scripts/04_ecl2_disulfide_analysis/` | Results III.5; Figure 6 | ECL2 depth and extracellular disulfide-region analyses. |
| `scripts/05_chemical_voxel_maps/` | Methods II.3; Results III.6; Figure 7 | Voxel-wise chemical maps and class-pair voxel export. |
| `scripts/06_cavity_geometry_helpers/` | Methods II.3; Results III.4 and III.6 | Geometry utilities used for cavity and voxel inspection. |
| `scripts/07_experimental_structure_controls/` | Results III.1; Supplementary Figures 1-2 | Control analyses with experimentally determined GPCR structures. |

`SCRIPT_INDEX.md` gives a script-by-script correspondence with the relevant part of the TFM.

## Data policy

This repository contains scripts only. Large structural datasets and generated results are not included: PDB collections, trajectory files, spreadsheets, figures, logs and compressed analysis outputs should be stored outside the repository or provided separately.

## Environment

The scripts were written for Python 3. The common Python dependencies are listed in `requirements.txt`.

```bash
pip install -r requirements.txt
```

Some workflows also require external resources:

- GPCRdb access for scripts that query receptor or structure annotations.
- PyMOL for PyMOL-specific visualization scripts.

## Reproducibility notes

Most scripts are command-line programs and expose their parameters through `--help`. Example:

```bash
python scripts/01_bw_annotation_and_distances/1_1_gpcr_distances_only.py --help
```

The scripts expect the corresponding input files described in the TFM methods, such as aligned GPCR PDB structures, `familias.txt`, volume text files, or intermediate Excel/CSV tables generated by previous analysis steps.
